Часто задаваемые вопросы

Найдите ответы на популярные вопросы о работе с генератором палитр

Для извлечения палитры из изображения:

  1. Перейдите на страницу "Генерация палитры"
  2. Нажмите на область загрузки или перетащите изображение
  3. Выберите количество цветов для извлечения (от 2 до 10)
  4. Дождитесь обработки изображения
  5. Получите готовую палитру с HEX-кодами

Поддерживаемые форматы: PNG, JPG, JPEG, WEBP

Генерация случайной палитры очень проста:

  1. На странице "Генерация палитры" найдите блок "Случайная палитра"
  2. Нажмите кнопку Сгенерировать случайную палитру
  3. Система создаст уникальную палитру из гармоничных цветов
  4. Вы можете повторно генерировать палитру, пока не найдете идеальный вариант
  5. Сохраните понравившуюся палитру в свой аккаунт
Совет: Используйте случайные палитры для вдохновения или когда нужны свежие идеи!

Сохранение палитры доступно только для зарегистрированных пользователей:

  1. Зарегистрируйтесь или войдите в аккаунт
  2. Сгенерируйте или извлеките палитру
  3. Нажмите кнопку Сохранить палитру
  4. Введите название для вашей палитры
  5. Подтвердите сохранение

Все сохраненные палитры доступны в разделе "Мои палитры".

Важно: Вы можете сохранять до 50 палитр в своем аккаунте.

Регистрация не обязательна для базовых функций:

Без регистрации
  • Генерация случайных палитр
  • Извлечение палитр из изображений
  • Экспорт палитр
  • Копирование HEX-кодов
С регистрацией
  • Сохранение палитр
  • Доступ к "Моим палитрам"
  • Управление коллекциями
  • Синхронизация между устройствами

Зарегистрироваться бесплатно

Базовые функции доступны без регистрации:

  • генерация случайных палитр;
  • извлечение палитр из изображений;
  • копирование HEX-кодов.

Регистрация нужна только для сохранения палитр и доступа к разделу "Мои палитры".
